Welcome to the documentation site for the assignment4shiny package.

Overview

The assignment4shiny package is designed to provide an interactive Shiny application and functions for data visualization and exploration, developed as part of an assignment for the ETC5523 unit.

Installation

remotes::install_github("ETC5523-2024/assignment4shiny")

Getting Started

Once installed, you can launch the Shiny app with the following commands:

library(assignment4shiny)
launchApp()
